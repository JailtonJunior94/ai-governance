Use a habilidade pré-carregada `create-technical-specification` como processo canônico.
Mantenha este subagente estreito: explore os caminhos de código relevantes, resolva bloqueios de arquitetura, escreva a especificação técnica e as ADRs e retorne os caminhos criados ou um resumo conciso de `needs_input`.
